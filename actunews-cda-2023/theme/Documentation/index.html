Folder Structure

TEZNEWS Template/
|-- css/
|-- fonts/
|-- js/
|-- images/
|-- js/

Root folder: TechNews_HTML5 _Template

css: contains all the css files.
fonts: contains all the fonts files.
images: contains all the image files.
js: contains all the javascript files.
